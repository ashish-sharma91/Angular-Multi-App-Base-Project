Golden Steps:
  1. ng g app-name
  2. ng g library shared-library -p fa
  3. ng g application contractor
  4. ng g application client
  5. ng g application employee
  6. ng serve
  7. ng serve client --port 4201
  8. ng serve contractor --port 4202
  9. ng serve employee --port 4203
  10. ng g c test-component --project=shared-library
  11. ng build shared-library
  12. npm install dist/shared-library
  13. import the library in ngModule
  14. Use component selector to use component in the desired application
